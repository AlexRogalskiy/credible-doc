Credible documentation

Maintaining of a project documentation is a complex task. We want to reduce efforts to accomplish this task.

The page contains a documentation of the tools created to achieve this goal.

Menu

  • Tools and libraries for replacing placeholders with a generated content
  • Utilities for generation of plantUML diagrams
  • Tool that allows to combine (merge) log files with different format of lines timestamps to a single file with lines sorted by timestamps
  • Example of a documentation generator

Other resources

  • A simple utility for packing and unpacking ISO 8583 messages
